# DG Smart Timetable

학교 교사들의 수업 교체와 보강 요청을 온라인에서 확인하고 처리하기 위한 정적 웹앱입니다.

배포 주소: https://rani0380.github.io/dgsmart_timetable/

## 현재 기능

- 교사별 시간표 조회
- 요일 및 학년 필터
- 교체/보강 요청 등록
- 승인, 반려, 대기 전환
- 동일 브라우저의 여러 탭 간 실시간 반영
- 변경 기록 및 충돌 감지
- Google Apps Script를 통한 구글 시트 자동 누적

## 데이터

`timetable-data.js`는 2026학년도 1학기 교사별 수업시간표 엑셀 파일에서 추출한 데이터입니다. 공개 저장소에 올릴 때는 교사 실명 포함 여부를 먼저 확인하세요.

## 구글 시트 연동

1. 구글 시트를 새로 만들고 `확장 프로그램 > Apps Script`를 엽니다.
2. `google-apps-script.gs` 내용을 Apps Script 편집기에 붙여넣고 저장합니다.
3. 스크립트 상단의 `ADMIN_EMAILS`에 담당자 이메일을 입력합니다.
4. `배포 > 새 배포 > 웹 앱`을 선택합니다.
5. 실행 권한은 본인, 액세스 권한은 학교 운영 방식에 맞게 설정합니다.
6. 배포 후 나온 웹 앱 URL을 `config.js`의 `googleSheetWebAppUrl`에 넣습니다.

URL이 비어 있으면 요청은 브라우저 로컬 저장소에만 저장됩니다.

요청이 접수되면 구글 시트에 누적되고, `ADMIN_EMAILS`에 입력된 담당자에게 이메일 알림이 발송됩니다.

구글 시트에 기록되지 않을 때는 Apps Script 편집기에서 `testAppendRequest_` 함수를 선택한 뒤 실행하세요. 테스트 행이 추가되면 Apps Script와 시트 연결은 정상이고, 웹 앱 배포 권한 또는 배포 버전을 확인하면 됩니다.

## 실행

`index.html`을 브라우저에서 열면 바로 실행됩니다.

## GitHub Pages 배포

이 프로젝트는 별도 빌드 과정이 없는 정적 사이트입니다. GitHub Pages에서 `main` 브랜치의 루트 폴더를 배포 대상으로 지정하면 됩니다.
