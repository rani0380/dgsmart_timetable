# DG Smart Timetable

학교 교사들의 수업 교체와 보강 요청을 온라인에서 확인하고 처리하기 위한 정적 웹앱입니다.

## 현재 기능

- 교사별 시간표 조회
- 요일 및 학년 필터
- 교체/보강 요청 등록
- 승인, 반려, 대기 전환
- 동일 브라우저의 여러 탭 간 실시간 반영
- 변경 기록 및 충돌 감지

## 데이터

`timetable-data.js`는 2026학년도 1학기 교사별 수업시간표 엑셀 파일에서 추출한 데이터입니다. 공개 저장소에 올릴 때는 교사 실명 포함 여부를 먼저 확인하세요.

## 실행

`index.html`을 브라우저에서 열면 바로 실행됩니다.
